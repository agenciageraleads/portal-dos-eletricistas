# /audit-ui - UI/UX & Quality Audit

$ARGUMENTS

## Task

Audit the frontend for UX compliance, accessibility standards, and React performance.

### Components

1. **UX Audit**: Checks against Nielsen's Heuristics and UI laws.
2. **Accessibility**: WCAG compliance check.
3. **React Performance**: Identifies unnecessary re-renders and inefficient patterns.
